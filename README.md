# YouTube Realtime

YouTube Realtime is a simple way to view your favorite channel's subscriber count updated in real time.

# Features
* YouTube Live Subscriber Count
* Embeds
* Compare Multiple Channel's Subscriber Count
* Themes
* **OBS SUPPORT**
